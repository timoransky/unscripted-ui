Dialog

Delete project?

This removes the project and all of its deployments. This can't be undone.

Native <dialog> + invoker commands (commandfor/command) to open declaratively, closedby="any" for light dismiss, @starting-style + allow-discrete for the entry/exit animation, ::backdrop for the blurred overlay. Focus trapping comes free.

Dropdown menu

popover attribute gives top-layer rendering, Esc + click-outside dismiss, and toggling with zero JS via popovertarget. CSS anchor positioning pins it under the trigger and auto-flips at viewport edges (position-try-fallbacks). Without anchor support it opens centered.

Accordion

Is it accessible?
Yes — <details>/<summary> ship with keyboard support and correct semantics, and the content is in the DOM for crawlers and find-in-page.
Can it animate height: auto?
Yes. interpolate-size: allow-keywords lets the ::details-content pseudo-element transition from height: 0 to height: auto. No measuring, no JS.
Only one open at a time?
The name attribute makes a group of details exclusive — the browser closes the others for you.

Select

appearance: base-select opts the real <select> into full styling: rich HTML inside options, ::picker(select) for the dropdown panel, <selectedcontent> mirrors the choice into the button, ::checkmark marks the selection. Chromium-only for now — everyone else gets the classic native select, which still works.
