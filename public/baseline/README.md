# Baseline logo assets

These SVGs are the official Baseline status marks (widely / newly / limited,
in light and dark variants). They are vendored here **unmodified**.

- Source: https://web-platform-dx.github.io/name-and-logo-usage-guidelines/
  (files under `https://web-platform-dx.github.io/assets/img/`)
- The Baseline name and logos are trademarks owned by Google.
- The logo assets are licensed under Creative Commons Attribution-NoDerivatives
  4.0 International (CC-BY-ND 4.0).

Per the usage guidelines, the colors are part of each logo and indicate the
feature's Baseline support level, so these files must not be recolored or
altered. Match the correct variant to the feature's status.
